Portfolio

I learned to design in code in 1999. Since then I've been fortunate to have a varied career in user experience design. The following are some examples of my work.

Wireframes

I have wireframed projects using a variety of software (OmniGraffle, Balsamiq, Illustrator) at both high and low fidelity. I like to use the wireframes to have an iterative conversation with the project stakeholders and quickly test design hypotheses before going to prototype.

Documentation

One of my major projects at NYPL was the conception and creation of the Library's first web properties style guide and rapid prototyping system (see the code at GitHub). Still a work in progress, the goals of this project were to produce a package of code to be used by developers that provides NYPL branding and a reference website that documented NYPL design patterns.

Design & Coding

For 8 years at the New York Public Library, I worked on the design and layout of NYPL.org, as well as other of its core web properties. (Note: most NYPL projects were designed as responsive.)
